The Problem

What does a newbie find when getting into web development for the first time?

If they're lucky, and young enough to be in school, they might learn web development on Adobe Dreamweaver, and we all know how that turns out:

We're looking at you MM_preloadImages().

If they're a little less lucky, they'll turn to the googles and search "learn html" or "html tutorial" and:

Get this unyielding pile of dung.

Or maybe they'll come accross a card carrying HTML expert on an internet forum who will tell them to:

Go read the spec.

This situation needs improvement.

The Plan

We (web developers) need to create a comprehensive "online book", with no paywall, which is fully crawlable by Google. It should cover the basics of HTML, CSS, and JavaScript, and include a full reference section for each.

So, I've started working on The HTML and CSS Tutorial (which currently only exists in the form of the web page you are staring at right now).

This online book will be tasty treat for the Google index, and – hopefully – link bait for the whole web development and design community. Each section and reference snippet should have a URL to be easily linked to from places like StackOverflow.

On top of that, there should be high quality video tutorials. Some of them for free, and others for sale.

A win for everyone, FTW.
